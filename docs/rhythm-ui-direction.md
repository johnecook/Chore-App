# Rhythm UI Direction

## Overall Style

Dark-first application.

Inspired by:

- Headspace
- Calm
- Linear
- Things
- Apple Health

Visual characteristics:

- Deep navy backgrounds
- Soft gradients
- Rounded corners
- Elevated cards
- Generous spacing
- High contrast typography

---

---

# Card Design

Border Radius:
20-24px

Cards should:

- Float slightly from background
- Use subtle shadows
- Use generous padding
- Avoid visible borders where possible

---

# Navigation

Bottom tab navigation.

Primary tabs:

- Today
- Tasks
- Goals
- Money
- More

---

# Child Experience

Focus on:

- Today's responsibilities
- Progress visibility
- Goal tracking
- Simplicity

No gamification.

No badges.

No levels.

No cartoon rewards.

---

# Parent Experience

Focus on:

- Family overview
- Pending approvals
- Activity feed
- Earnings
- Goal progress

---

# Accessibility

Must support:

- Large iOS Dynamic Type
- Dark Mode
- High Contrast
- Screen Readers
- Color Blind Accessibility

---

# Motion

Animations should be:

- Subtle
- Smooth
- Purposeful

Avoid:

- Bouncing
- Flashing
- Confetti
- Excessive celebration animations
